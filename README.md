# 💍🌙 MoonStone- Online Jewelry Store

## Project Description
MoonStone is a web-based e-commerce platform for buying and selling jewelry. 
Users can browse collections, create accounts, purchase products, and track their orders.
Admins can manage inventory, users, and orders.

## Tech Stack
- Backend: Spring Boot
- Frontend: HTML, CSS, Bootstrap
- Security: Spring Security
- Database: MySQL
- Deployment: Docker

## User Stories

**Story 1 - Customer Registration & Login:**
As a new customer, I want to create an account and log in securely so that I can purchase jewelry and view my order history.

**Story 2 - Product Browsing & Purchase:**
As a customer, I want to browse jewelry by category (rings, necklaces, earrings), add items to my cart, and complete a purchase.

**Story 3 - Admin Product Management:**
As an admin, I want to add new jewelry products with photos and prices, update stock quantities, and manage customer accounts.
