
# 💍🌙 MoonStone — Online Jewelry Store

**Project Description**
MoonStone is a web-based e-commerce platform for buying and selling luxury jewelry. Users can browse collections by category, create accounts with JWT-secured authentication, manage their cart and wishlist, complete purchases, book personal jewelry consultations, and track their orders. Admins can manage the full product inventory and user accounts.

---

**Tech Stack**

* **Backend:** Spring Boot (REST API)
* **Frontend:** HTML, CSS, Bootstrap (static pages served via Spring)
* **Security:** Spring Security + JWT (Stateless, BCrypt password hashing)
* **Database:** MySQL (via Spring Data JPA / Hibernate)
* **Build Tool:** Maven

---

**User Stories**

**Story 1 — Customer Registration & Login**
As a new customer, I want to register with my name, email, and password and log in securely using JWT-based authentication, so that I can access my personal cart, wishlist, and order history.

**Story 2 — Product Browsing & Purchase**
As a customer, I want to browse jewelry by category (Rings, Necklaces, Earrings, Bracelets, Brooches, Pendants), view product details with tags like "New", "Sale", or "Exclusive", add items to my cart or wishlist, and complete a purchase that creates a tracked order.

**Story 3 — Jewelry Consultation Appointment**
As a customer, I want to book a personal jewelry consultation by selecting a service type, preferred advisor, date, and available time slot, so that I can receive expert guidance before making a high-value purchase.

**Story 4 — Admin Product Management**
As an admin, I want to add new jewelry products with photos, descriptions, categories, price, stock quantity, and promotional tags, update existing listings, and manage customer accounts and orders across the platform.
